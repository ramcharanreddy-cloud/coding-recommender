# 🚀 AI Coding Recommendation System

This project is a Machine Learning-based web application that recommends the most suitable programming language based on user input such as interest, skill level, and goals.

## 🔥 Features
- 🎯 Smart language recommendation using ML
- 📊 Confidence-based predictions
- 🌐 FastAPI backend for model serving
- 💻 Simple web interface with dropdown inputs
- 🔄 Real-time prediction

## 🧠 Technologies Used
- Python
- Scikit-learn (Logistic Regression)
- FastAPI
- HTML, CSS, JavaScript

## ⚙️ How It Works
1. User selects preferences (Interest, Skill Level, Goal)
2. Frontend sends request to FastAPI backend
3. Backend processes input and feeds into trained ML model
4. Model predicts best programming language
5. Result is displayed on the webpage

## 📂 Project Structure
- `src/` → ML logic (training, prediction)
- `models/` → Saved model and encoders
- `app/` → FastAPI backend
- `index.html` → Frontend UI
